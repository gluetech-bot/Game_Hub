\section{Introduction}
In this project, we developed a mini game hub using Pygame, a popular Python library for game development. The game hub includes three different games: Tic Tac Toe, Connect 4, and Othello. Each game is designed to provide an engaging and enjoyable gaming experience while showcasing our programming skills and creativity.

\section{Design and Architecture}
There are three main components of the game hub: the main menu, the individual games, and the leaderboard system. The main menu serves as the central hub where players can select which game to play. Each game has its own unique interface and gameplay mechanics, while the leaderboard system keeps track of player scores and rankings across all games.
\subsection{Tic Tac Toe}
Tic Tac Toe is a classic strategy game genrally played on a 3x3 grid,but this tic tac toe is 10x10. Two players take turns placing their marks (X or O) on the grid, with the goal of getting five of their marks in a row (horizontally, vertically, or diagonally) before the opponent does.
\subsection{Connect 4}
Connect 4 is a two-player connection game in which players take turns dropping colored discs into a vertically oriented grid. The objective is to be the first player to get four of their own discs in a row (horizontally, vertically, or diagonally).
\subsection{Othello}
Othello, also known as Reversi, is a strategy board game for two players. The game is played on an 8x8 grid, and players take turns placing their pieces on the board. The objective is to have the most pieces on the board by the end of the game, which is achieved by surrounding and flipping the opponent's pieces.

\section{Features Implemented}

    \item Individual game interfaces with unique gameplay mechanics
    \item Leaderboard system to track player scores and rankings

\section{Running Instructions}
    \subsection{Prerequisites}
    \begin{enumerate}
        \item Ensure you have Python installed on your system.
        \item Install the required libraries using pip:
        \begin{verbatim}
            pip install pygame-ce numpy matplotlib
            pip install numpy
        \end{verbatim}
        If it's showing error like "This environment is externally managed" then:
        \begin{verbatim}
            sudo apt install python3-xyz
        \end{verbatim}
        , where xyz is the package you are trying to install.
        \item Navigate to the project directory in your terminal.
        
        \end{enumerate}

\section{Retrosepective}
    This project was a great learning experience for us.
    We faced several challenges during the development process,
    such as implementing the game logic, designing the user interface,
    and managing user accounts and leaderboards. However,
    we overcame these challenges through research,experimentation,
    and collaboration. We also learned how to use Pygame effectively and gained insights into game development principles.\\
    Some specific challenges we faced include:\\
    \begin{itemize}
        \item There were some tricky bugs in taking user input and making sure that there are no spaces that can cause trouble in leaderboard and user management.
        \item Designing a user-friendly interface that was visually appealing and easy to navigate.
        \item Preparing the end loop,when to give graphs.Where to prompt play again etc.
    \end{itemize}
    We brain stormed and implemented various solutions to fix the bugs which can possbly occur.
    We also designed the interface iteratively, taking feedback from each other and making improvements based on that feedback.\\
    Overall, this project allowed us to apply our programming skills in a practical context and provided us with valuable experience in game development, user interface design, and problem-solving.\\
    With some more time, we would have liked to add more games to the hub,make some changes to UI.
